# Get Brazilian cities table

Returns the Brazilian cities table with codes and names.

## Usage

``` r
comex_cities(verbose = FALSE)
```

## Arguments

- verbose:

  Logical. Default: `FALSE`.

## Value

A data.frame with city (IBGE) codes and names.

## Examples

``` r
if (FALSE) { # \dontrun{
comex_cities()
} # }
```
