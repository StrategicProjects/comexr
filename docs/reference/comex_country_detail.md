# Get country details

Returns details for a specific country by its code.

## Usage

``` r
comex_country_detail(id, verbose = FALSE)
```

## Arguments

- id:

  Country code (e.g. `105` for Brazil).

- verbose:

  Logical. Default: `FALSE`.

## Value

A list with country details.

## Examples

``` r
if (FALSE) { # \dontrun{
comex_country_detail(105)
} # }
```
