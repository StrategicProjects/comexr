# Get Brazilian states (UF) table

Returns the Brazilian states table with codes and names.

## Usage

``` r
comex_states(verbose = FALSE)
```

## Arguments

- verbose:

  Logical. Default: `FALSE`.

## Value

A data.frame with state codes and names.

## Examples

``` r
if (FALSE) { # \dontrun{
comex_states()
} # }
```
