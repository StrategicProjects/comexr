# Get state details

Returns details for a specific Brazilian state.

## Usage

``` r
comex_state_detail(uf_id, verbose = FALSE)
```

## Arguments

- uf_id:

  State code (e.g. `26` for Pernambuco).

- verbose:

  Logical. Default: `FALSE`.

## Value

A list with state details.

## Examples

``` r
if (FALSE) { # \dontrun{
comex_state_detail(26)
} # }
```
