# Get NBM code details

Returns details for a specific NBM code.

## Usage

``` r
comex_nbm_detail(nbm_code, verbose = FALSE)
```

## Arguments

- nbm_code:

  NBM code (e.g. `"2924101100"`).

- verbose:

  Logical. Default: `FALSE`.

## Value

A list with NBM details.

## Examples

``` r
if (FALSE) { # \dontrun{
comex_nbm_detail("2924101100")
} # }
```
