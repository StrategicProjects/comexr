# Get city details

Returns details for a specific Brazilian city.

## Usage

``` r
comex_city_detail(city_id, verbose = FALSE)
```

## Arguments

- city_id:

  IBGE city code (e.g. `5300050`).

- verbose:

  Logical. Default: `FALSE`.

## Value

A list with city details.

## Examples

``` r
if (FALSE) { # \dontrun{
comex_city_detail(5300050)
} # }
```
